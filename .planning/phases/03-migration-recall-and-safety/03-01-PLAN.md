**cmd/migrate.sh** — Create new executable file (`chmod +x`):

1. Shebang + `set -euo pipefail`. Header comment: "Hot-to-cold migration for stale models".
2. SCRIPT_DIR resolution via BASH_SOURCE. Source: common.sh, config.sh, hf_adapter.sh, ollama_adapter.sh, audit.sh.
3. Parse args:
   - `DRY_RUN=false; [[ "${1:-}" == "--dry-run" ]] && DRY_RUN=true`
   - `TRIGGER="manual"; [[ "${TRIGGER_SOURCE:-}" == "cron" ]] && TRIGGER="cron"`
4. `load_config` to get HOT_HF_PATH, HOT_OLLAMA_PATH, COLD_PATH, RETENTION_DAYS.
5. Constants: `USAGE_FILE="${HOME}/.modelstore/usage.json"`, `OP_STATE_FILE="${HOME}/.modelstore/op_state.json"`.
6. State file helpers:
   - `_write_op_state() { local op="$1" model="$2" phase="$3" trigger="$4"; jq -cn --arg op "$op" --arg m "$model" --arg ph "$phase" --arg ts "$(date -u +%Y-%m-%dT%H:%M:%SZ)" --arg tr "$trigger" '{op:$op,model:$m,phase:$ph,started_at:$ts,trigger:$tr}' > "${OP_STATE_FILE}.tmp" && mv "${OP_STATE_FILE}.tmp" "$OP_STATE_FILE"; }`
   - `_clear_op_state() { rm -f "$OP_STATE_FILE"; }`
   - Stale state file cleanup: if `$OP_STATE_FILE` exists, read `started_at`, compute age. If older than 4 hours (14400 seconds), log warning "Clearing stale operation state (started at ...)" and `_clear_op_state`. If not stale, log "Resuming interrupted operation" and continue.
7. `find_stale_hf_models()`:
   - Read USAGE_FILE, compute cutoff_epoch from RETENTION_DAYS.
   - For each entry in usage.json where key starts with HOT_HF_PATH/models--: skip if symlink (`[[ -L "$model_path" ]]`), check if last_epoch < cutoff_epoch, echo path.
   - Also walk `${HOT_HF_PATH}/models--*/` for dirs NOT in usage.json (never tracked = stale). Skip symlinks.
8. `find_stale_ollama_models()`:
   - Similar logic but for Ollama models. Keys in usage.json matching HOT_OLLAMA_PATH. Also walk manifest dirs for untracked models.
9. Dry-run rendering (`$DRY_RUN == true`):
   - Collect stale HF + Ollama models into arrays.
   - Print "Would migrate" table with columns: MODEL, SIZE, LAST USED, DAYS AGO, ACTION. Use `printf` for alignment.
   - Print total size and count.
   - Print "Keeping hot" table: models NOT stale, with REASON column ("used within N days" or "already on cold").
   - Print cold store available space: `df -BG --output=avail,size "$COLD_PATH"`.
   - Exit 0 without touching any data.
10. Real migration (`$DRY_RUN == false`):
    - Check cold mounted: `check_cold_mounted "$COLD_PATH"`.
    - For each stale HF model:
      - `_write_op_state "migrate" "$model" "rsync" "$TRIGGER"`.
      - Record start time.
      - Call `hf_migrate_model "$model" "$COLD_PATH"`. Capture return code.
      - `_write_op_state "migrate" "$model" "cleanup" "$TRIGGER"`.
      - Record end time, compute duration.
      - Get model size via `hf_get_model_size` (before migration) or `du -sb`.
      - `audit_log "migrate" "$model" "$size" "$model" "${COLD_PATH}/hf/$(basename "$model")" "$duration" "$TRIGGER"` on success.
      - `audit_log "fail" "$model" "$size" "$model" "" "$duration" "$TRIGGER" "migration failed"` on failure.
      - `_clear_op_state`.
    - For each stale Ollama model:
      - Same pattern: _write_op_state, call ollama_migrate_model, audit_log, _clear_op_state.
    - Log summary: "Migration complete: N models moved".
