**modelstore/lib/common.sh** — Shared safety and logging functions. Must contain:

1. Source lib.sh using BASH_SOURCE (not $0):
```bash
_TOOLBOX_LIB="$(cd "$(dirname "${BASH_SOURCE[0]}")/../../" && pwd)/lib.sh"
source "$_TOOLBOX_LIB"
```

2. `ms_log()` — Log to stderr with `[modelstore]` prefix: `echo "[modelstore] $*" >&2`

3. `ms_die()` — Log error and exit 1: `echo "[modelstore] ERROR: $*" >&2; exit 1`

4. `check_cold_mounted()` — Takes one arg (cold_path). Uses `mountpoint -q "$cold_path"` and calls `ms_die` on failure. NOT `test -d`.

5. `check_space()` — Takes destination_path and required_bytes. Uses `df --output=avail -B1 "$dest" | tail -1` to get available bytes. Applies 10% safety margin: `available * 90 / 100`. Returns 1 if insufficient, 0 if OK. Prints warning via ms_log on failure.

6. `validate_cold_fs()` — Takes one arg (cold_path). Uses `findmnt --output FSTYPE --target "$cold_path" --noheadings`. Accepts ext4, xfs, btrfs (return 0). Rejects exfat, vfat, ntfs with clear error message explaining symlinks require a Linux filesystem and suggesting `sudo mkfs.ext4 /dev/sdX`. Returns 1 on rejection. Unknown filesystem: print warning but return 0.

Shebang: `#!/usr/bin/env bash`. No `set -e` in lib files (caller controls that). All variable refs quoted.

**modelstore/lib/config.sh** — JSON config read/write via jq. Must contain:

1. `MODELSTORE_CONFIG="${HOME}/.modelstore/config.json"` — single constant for config path.

2. `config_exists()` — `[[ -f "$MODELSTORE_CONFIG" ]]`

3. `config_read()` — Takes one arg (jq key like `.hot_hf_path`). Runs `jq -r "$1" "$MODELSTORE_CONFIG"`. Returns the value.

4. `load_config()` — Checks `config_exists`, if false prints `"modelstore: not initialized. Run: modelstore init"` to stderr and exits 1. Otherwise sets these variables: `HOT_HF_PATH`, `HOT_OLLAMA_PATH`, `COLD_PATH`, `RETENTION_DAYS`, `CRON_HOUR` by calling `config_read` for each.

5. `write_config()` — Takes 6 positional args: hot_hf, hot_ollama, cold, retention_days, cron_hour, backup_retention_days. Uses `jq -n` with `--arg` and `--argjson` to produce the full JSON schema (version: 1, created_at/updated_at via `now | todate`). Writes to `$MODELSTORE_CONFIG` with `chmod 600`.

6. `backup_config_if_exists()` — If config exists, copies to `${MODELSTORE_CONFIG}.bak.$(date +%Y%m%dT%H%M%S)` with chmod 600. Then cleans up backups older than `backup_retention_days` (read from existing config, default 30) using `find ... -name "config.json.bak.*" -mtime "+${retention_days}" -delete`.

Shebang: `#!/usr/bin/env bash`. No `set -e` in lib files. No side effects on source (no echo, no mkdir, no exit).

- `#!/usr/bin/env bash` + `set -euo pipefail`
