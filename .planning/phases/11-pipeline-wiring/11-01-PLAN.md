**scripts/screen-data.sh** — Standalone training data pre-screening script.

Header: `#!/usr/bin/env bash` with `set -euo pipefail`

Usage: `screen-data.sh <input_file> [output_dir]`
- `input_file`: path to .jsonl, .txt, or .parquet file
- `output_dir`: defaults to same directory as input_file

Environment variables:
- `HARNESS_URL` (default: `http://localhost:5000`)
- `HARNESS_API_KEY` (required, must be a non-bypass tenant key like dev-team)

Workflow:
1. Parse args, validate input file exists
2. Detect file extension (.jsonl, .txt, .parquet). For .parquet, check `python3 -c "import pandas"` — exit with clear error if unavailable
3. Health check: `curl -sf "${HARNESS_URL}/health" >/dev/null 2>&1 || curl -sf "${HARNESS_URL}/" >/dev/null 2>&1`. If both fail, print "ERROR: Harness not reachable at ${HARNESS_URL}. Start harness first." and exit 1
4. Set up output files: `${output_dir}/${basename}-screened.${ext}` and `${output_dir}/${basename}-removed.log`
5. Initialize counters: TOTAL=0, CLEAN=0, FLAGGED=0
6. Read records:
   - .txt: each non-empty line is one record
   - .jsonl: each line is JSON; extract the text content (look for "text", "prompt", or "content" field via python3 one-liner)
   - .parquet: use `python3 -c "import pandas; df = pandas.read_parquet('$INPUT'); [print(row) for row in df.iloc[:,0]]"` to extract first column as text records
7. For each record, POST to `${HARNESS_URL}/v1/chat/completions` with:
   ```
   curl -sf -o /dev/null -w "%{http_code}" \
     -H "Authorization: Bearer ${HARNESS_API_KEY}" \
     -H "Content-Type: application/json" \
     -d '{"model":"screen","messages":[{"role":"user","content":"'"${ESCAPED_RECORD}"'"}]}' \
     "${HARNESS_URL}/v1/chat/completions"
   ```
   HTTP 200 = clean record, write to screened output.
   HTTP 400/403/422 = flagged, write to removed log with reason (capture response body).
   Other codes = treat as error, log and skip.
8. Print summary: "Screened: TOTAL total, CLEAN clean, FLAGGED removed. See ${removed_log}"
9. Exit 0 if any clean records remain, exit 1 if all records were flagged

Important: Escape record content for JSON embedding — use python3 for JSON string escaping: `python3 -c "import json,sys; print(json.dumps(sys.stdin.read().strip()))"`. Do NOT use bash string manipulation for JSON escaping.
